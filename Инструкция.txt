Для функционирования необходимо: Node.js


 Развертывание и запуск:
	1. Открыть окно консоли в корневой папке проекта.
	2. Ввести комманду " npm install "
	3. После загрузки и установки всех компонентов ввести " node app.js "
	4. По умолчанию сайт будет доступен по адресу: localhost:3000
  
 Состав:
	app.js - "главный" файл
	bin - папка сожержащая разнообразные настройки
	package.json - файл содержащий информацию о зависимостях
	Procfile - содержит информацию для запуска приложение на хостинге Heroku
	Public - папка содержащая общедоступные файлы - скрипты, стили, изображения.
  	Routes - папка содержит файлы отвечающие за роутинг и обработку запросов.
	Views - папка содержит верстку, папка layouts содержит часть верстки кторая преобразуется в верхнюю и
	нижнюю части сайта. Так же там размещается head и подключаются все стили и скрипты.
	
Примечания: 
	* Для рассылки писем используется сервис "Mailgun", а именно бесплатный план сервиса (ограничен 10 000
	  писем/месяц )
	* Если вы хотите запускать сайт в продакшн рекомендуется создать новый аккаунт на "Mailgun" и исполь-
	  зовать платный план.
	* Все что связано с "Mailgun" находится в: routes/index.js
